Οδηγος Git

Αρχική

mkdir (και το ονομα του φακελου που δημιουργουμε) για να γινει ο πρωτος φακελος.

cd (και το ονομα του αρχειου) για να μπουμε μεσα στον φακελο που δημιουργησαμε.

git init για να ενεργοποιησουμε τον φακελο ωστε να καταλαβει το git οτι ειναι ενα καινουργιο project.

touch (με το ονομα του πρωτου αρχειου που θα μπει στον φακελο με την καταληξη .html) για να δημιουργουμε το πρωτο μας αρχειο μεσα στον φακελο.

git status για να δουμε τα αρχεια που βρισκονται μεσα στον φακελο μας.

git add . για να προσθεσουμε ολα τα αρχειο στον φακελο μας.

git commit -m "Και ενα μηνυμα" για να αποθηκευσουμε τις αλλαγες μας στο local repository.

git push για να μεταφερουμε το τελευταιο commit που καναμε στον server του github
