Οδηγος Git

Αρχική

mkdir (και το ονόμα του φακέλου που δημιουργούμε): για να γίνει ο πρώτος φάκελος.

cd (και το όνομα του αρχείου): για να μπούμε μέσα στον φάκελο που δημιουργήσαμε.

git init: για να ενεργοποιήσουμε τον φάκελο, ώστε να καταλάβει το git οτι είναι ενα καινούργιο project.

touch (με το όνομα του πρώτου αρχείου που θα μπει στον φάκελο με την κατάληξη ".html"): για να δημιουργούμε το πρώτο μας αρχείο μεσα στον φάκελο.

git status: για να δούμε τα αρχεία που βρίσκονται μέσα στον φάκελό μας.

git add . : για να προσθέσουμε όλα τα αρχεία στον φάκελο μας.

git commit -m "Και ενα μηνυμα": για να αποθηκεύσουμε τις αλλαγές μας στο local repository.

git push: για να μεταφέρουμε το τελευταίο commit που κάναμε στον server του github
